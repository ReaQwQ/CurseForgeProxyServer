# CurseForge Proxy Server

セキュアなCurseForge API管理のためのプロキシサーバー

## 使用方法

詳細なセットアップ手順と使用方法は[README.md](README.md)を参照してください。

## クイックスタート

```bash
# 依存関係のインストール
npm install

# .envファイルを作成
cp .env.example .env

# .envにCurseForge APIキーを設定
# CURSEFORGE_API=your_api_key_here

# サーバーを起動
npm start
```
